# 📉 A股自选股智能分析系统 - 使用说明

## 1. 环境准备

本系统基于 **Python** 开发，请确保您的电脑已安装 Python 环境。

- **推荐版本**: Python 3.8 或以上
- **下载地址**: [Python 官网](https://www.python.org/downloads/)

> **注意**: 安装 Python 时，请务必勾选 **"Add Python to PATH"** (将 Python 添加到环境变量)。

---

## 2. Windows 用户使用指南

### 第一步：一键安装
找到文件夹中的 `install.bat` 文件，**双击运行**。
- 脚本会自动创建虚拟环境。
- 自动下载并安装所需的依赖库（使用国内镜像，速度快）。
- 如果看到 "安装完成" 的提示，说明环境准备好了。

### 第二步：启动程序
找到文件夹中的 `run.bat` 文件，**双击运行**。
- 程序会自动启动。
- 稍等几秒，浏览器会自动打开系统界面 (通常是 http://localhost:8000)。

---

## 3. Mac / Linux 用户使用指南

### 方法一：直接运行 (推荐)
找到文件夹中的 `start_direct.command` 文件，**双击运行**。
- 如果提示权限不足，请在终端执行：`chmod +x start_direct.command`。
- 脚本会自动检查环境、安装依赖并启动 Web 界面。

### 方法二：终端运行
打开终端，进入项目目录：

```bash
# 1. 创建并激活虚拟环境
python3 -m venv venv
source venv/bin/activate

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动程序
python src/main.py --webui-only
```

## 4. 常见问题

**Q: 双击 `install.bat` 闪退怎么办？**
A: 这通常是因为电脑没有安装 Python，或者没有配置环境变量。请重新安装 Python 并勾选 "Add to PATH"。

**Q: 安装依赖时提示网络错误？**
A: 脚本默认使用了清华大学的镜像源。如果依然失败，请检查网络连接，或者稍后重试。

**Q: 启动后浏览器显示 "无法连接"？**
A: 请等待黑色命令行窗口中出现 "Uvicorn running on..." 字样后再刷新页面。如果端口 8000 被占用，程序可能会报错，请检查是否有其他程序占用了该端口。

**Q: 如何修改自选股？**
A: 在 Web 界面中可以直接添加或删除自选股。也可以在项目目录下的 `.env` 文件中修改 `STOCK_LIST` 配置（如果文件不存在，可以将 `.env.example` 复制一份重命名为 `.env`）。
